== Description ==

这是一款解决把你文章内容推送到今日头条搜索里，让头条搜索第一时间抓取你的内容，并且支持扫描死链接提交到头条搜索.

## 插件功能
* 支持文章 嵌入今日头条搜索自动收录代码
* 支持新增扫描死链接提交到头条搜索+
* 支持扫描出永久删除文章
* 支持扫描出回收站的文章
* 支持后台自定义嵌入代码
* 支持对应https
